# repository_template
公開リポジトリのテンプレート

このテンプレートを使うことで次の項目が自動設定されます。

- READMEのテンプレート（本ファイル）
- Contributing Guidline （未実装）
- issue template
- pull request template（未実装）

# How To Use

## テンプレートの適用時
リポジトリ作成画面で**Repository template**に`rt-net/repository_template`を設定します。

## テンプレート適用後

以降の手順はリポジトリの種類に合わせて適宜変更してください。

- 本README.mdを書き換える
